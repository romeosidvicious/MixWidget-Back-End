MixWidget-Back-End by romeosidvicious

A php backend for MixWidget

MixWidget Back End is a drop in back end to allow for the easy creation of flash based mix tapes. It is designed to be used with a single login, currently .htaccess based, on a website. It generates two pages for each mix the first being used for display on the public index page, which displays all of the mixes created, and the second page which is complete html and can be used for showing off a single mix of copying and pasting from the source to post the mix on a blog or other website. Options to present code for embedding and for presenting a link to an archive of all of the songs included in the mix are present as well as some cover art functionality.

There is still a lot of work to be done but the functionality is stable. The interface is usable but not pretty. Currently no options exist for editing mixes outside of deleting mixes that have been created.

The actual flash widget is not part of this project and available at: http://mixwidget.org/ The skins included can be acquired from the above URL and are provided by: http://www.tapedeck.org/

Dependencies

PHP 5.2 getid3 (included)

Install

These instructions cover installation only. After install documentation is provided on using 
the MixWidget Back End.

NOTE: If your webserver does not support .htaccess and .htpasswd you MUST secure the 
admin/ directory in some manner. If you do not anyone will be able to access this script and upload 
files to your server.


Shell:

Upload the MWBE tar.gz to your server.
Login in to your shell and extract MWBE into whatever directory you wish.
Change the admin password in admin/.htpasswd using .htpasswd

Make the following files and dirs world readable/writable (chmod 777):
whatever dir you uploaded MWBE to
archives/
confs/
mixes/
playlists/
tracks/
mixes.php

Point your browser to http://www.yourdomain.com//admin
Start making mix tapes

FTP:

Extract the MWBE tar.gz to an empty directory on your hard drive
Go to http://www.clockwatchers.com/htaccess_tool.html and scroll down to 2. .htpasswd Tool
Put in a username and password and click Create It
Go to the admin directory in the directory you just extracted the files and open .htpasswd with 
a text editor (notepad NOT wordpad)
Delete the current username and password in .htpasswd and paste in the results from 
clockwatchers.com
Use your FTP client to upload all of the files to an empty directory on your webserver
Use your FTP client to make sure the following files and directories are readable and writable 
by everyone (world or 777 permissions):
whatever dir you uploaded MWBE to
archives/
confs/
mixes/
playlists/
tracks/
mixes.php

Point your browser to http://www.yourdomain.com//admin
Start making mix tapes
